# CET-English-test-registration-system
## 简介

基于SpringBoot的四六级报名系统  包括学生用户与管理员用户。

## 使用技术

1. 框架：SpringBoot
2. 项目管理：Maven
3. 数据库：Mysql
4. 前端：Vue+Element
